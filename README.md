# Site da Seleção Brasileira

Este é um site simples feito com HTML e CSS sobre a Seleção Brasileira.

## Tema

O site fala sobre o Brasil na Copa do Mundo 2026.

## O que tem no site

- Resumo da história da Seleção Brasileira
- Títulos conquistados pelo Brasil
- Lista de jogadores convocados
- Jogos do Brasil na Copa
- Visual com as cores da bandeira do Brasil
